Usage

Overview

Proxy Helper switches Chrome's proxy independently of the system proxy. Each profile holds a complete proxy configuration; pick a profile and a proxy type from the popup to apply it.

Profiles

Create one profile per environment (Work, Home, Travel). Use the bar at the top of this page: + New to create, Rename / Duplicate / Delete to manage. Switch the active profile from the popup.

Switching Profiles

Click the toolbar icon to open the popup. Click a profile to expand it, then click a proxy type — that profile becomes active and the proxy is applied to Chrome immediately.

Proxy Types

Each profile remembers which proxy type it's using. The popup lists Direct plus every HTTP / HTTPS / SOCKS / PAC entry you've filled in under the other tabs; System and Auto Detect (WPAD) sit behind a "More" row.

Server Configuration

Enter proxy host and port for HTTP, HTTPS, or SOCKS in the General tab. For SOCKS, pick the protocol version (SOCKS4 or SOCKS5). Proxy Mode controls how Chrome routes requests across proxies (singleProxy uses one proxy for all traffic).

PAC (Proxy Auto-Config)

PAC scripts decide the proxy per URL. Paste a script inline under PAC Script, or point to a hosted script under PAC URL.

Bypass List

Domains and IP ranges that should not use the proxy. Separate entries with commas. Example: <local>, 192.168.0.0/16, *.intranet.com. Enable China List to also bypass *.cn domains.

Authentication

Username and password for proxies that require basic authentication. Note: Chrome's API supports authentication for HTTP/HTTPS proxies only, not SOCKS.

Keyboard Shortcuts

Alt+P cycles to the next profile. Alt+O opens this options page. Customize the shortcuts at chrome://extensions/shortcuts.
